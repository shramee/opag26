# SDK

Reusable TypeScript utilities packaged for Node.js, browser, and React applications.

## Install

```bash
npm install
```

## Usage

```ts
import { add } from "@opag26/sdk";

const total = add(2, 3);
```

## Scripts

- `npm run build` builds ESM, CJS, and type declarations into `dist/`
- `npm test` runs the Vitest suite
